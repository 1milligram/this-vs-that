## Running it on local

1. Clone the project:

~~~ console
$ git clone https://github.com/phuoc-ng/this-vs-that
~~~

2. Install the dependencies:

~~~ console
$ cd this-vs-that
$ npm install
~~~

3. Run it on the local:

~~~ console
$ npm run dev-server
~~~

Open http://localhost:1234 in your browser to see it in action.
