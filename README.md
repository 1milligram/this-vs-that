## Running it on local

1. Clone the project:

```shell
$ git clone https://github.com/phuocng/this-vs-that
```

2. Install the dependencies:

```shell
$ cd this-vs-that
$ npm install
```

3. Run it on the local:

```shell
$ npm run start
```

Open http://localhost:8081 in your browser to see it in action.
