## Running it on local

1. Clone the project:

```shell
$ git clone https://github.com/1milligram/this-vs-that
```

2. Install the dependencies:

```shell
$ cd this-vs-that
$ npm install
```

3. Run it on the local:

```shell
$ npm run dev
```

Look at the console to see the _Local URL_. Open it in your browser to see it in action.
