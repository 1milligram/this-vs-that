## Running it on local

1. Clone the project:

```shell
$ git clone https://github.com/phuoc-ng/this-vs-that
```

2. Install the dependencies:

```console
$ cd this-vs-that
$ npm install
```

3. Run it on the local:

```console
$ npm run serve
```

Look at the console to see the _Local URL_. Open it in your browser to see it in action.
